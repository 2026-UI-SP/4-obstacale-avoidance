# Obstacle Avoidance
The Object Avoidance application is a research focused application with slightly moderated access from the client, Dr. Flint. After the User submits a demographic survey provided by Dr. Flint, they will receive a Participant ID used to begin participation in the application. Upon submitting a valid ID, the participant is provided a brief description of the task they are to complete, and when ready, will select a start button.
